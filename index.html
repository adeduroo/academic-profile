Hybrid-Based Cryptographic Algorithm for Cloud Data Security

Ph.D. Research | Babcock University

This research developed a novel hybrid cryptographic algorithm that combines symmetric and asymmetric encryption techniques to enhance cloud data security. The algorithm addresses key challenges in cloud storage including data privacy, access control, and secure data sharing.

Key Contributions:

  • Proposed a hybrid encryption model that optimizes performance and security
  • Implemented a dynamic key management system for enhanced security
  • Conducted extensive security analysis and performance evaluation
  • Published findings in peer-reviewed journals
Evaluation of Biometric Authentication Scheme

Undergraduate Research | FUT Akure

This research evaluated various biometric authentication schemes to determine their effectiveness, security, and usability in different application contexts.

Research Focus:

  • Comparative analysis of fingerprint, iris, and facial recognition systems
  • Evaluation of false acceptance and rejection rates
  • Security vulnerability assessment
  • Usability testing in different environments
  • Recommendations for implementation in secure systems
